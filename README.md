# NLW Return Server
O backend construído do app de feedbacks feito na NLW, com Node (Express), SOLID, Prisma e Jest para testes unitários.

### Como Executar o Projeto
```bash
# INSTALAR DEPENDÊNCIAS
yarn install

# RODAR O AMBIENTE DE DEV
yarn dev

# RODAR TESTES
yarn test
```
